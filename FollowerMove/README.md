# 簡介
跟隨者移動參數的調整。

# 插件參數
| 名稱 | 型別 | 預設值 | 說明 |
|:----------------------|:------------------|:--|:----|
| follower speed        | [1, 6]Z:Number    | 4 | 跟隨者的移動速度 |
| follower frequency    | [1, 5]Z:Number    | 5 | 跟隨者的移動頻率 |
| follower collid       | [0, 1]Z:Boolean   | 1 | 跟隨者的是否允許碰撞 |
